## How to Get Your Free Groq API Key

1. **Sign up at Groq**: https://console.groq.com
2. **Get your API key**:
   - Go to "API Keys" section
   - Click "Create API Key"
   - Copy the key (starts with `gsk_...`)

3. **Add it to your portfolio**:
   - Log in with Discord on your portfolio
   - Go to the **Settings Module** node
   - Paste your API key
   - Click "Save Configuration"

4. **Start chatting**:
   - Go to the **AI Assistant** node
   - Ask anything!

## Troubleshooting

**"API Error" message in chat?**
- Make sure you added your Groq API key in Settings
- Check console (F12) for detailed error messages
- Verify your API key is valid at console.groq.com

**Premium nodes not appearing?**
- Make sure you're logged in with Discord
- Refresh the page after logging in
- Check browser console for errors

**Want to test locally?**
- The chatbot works on localhost too!
- Just log in with Discord and add your API key
